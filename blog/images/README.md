# 博客图片文件夹

这个文件夹用于存放博客文章的封面图片。

## 图片命名规范

- 文章封面图片命名格式：`postX.jpg`（X为文章编号）
- 推荐图片尺寸：800x400像素
- 支持格式：jpg, png, webp

## 注意事项

- 如果没有为文章设置图片，系统会自动显示带图标的占位符
- 图片文件不存在时，会自动隐藏图片区域
- 建议使用压缩后的图片以提高加载速度
